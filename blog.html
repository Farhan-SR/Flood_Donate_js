What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each element (like tags, text, attributes) is a node. Using the DOM, programming languages like JavaScript can interact with and manipulate the content, structure, and styles of a webpage dynamically.

How do you select an element from DOM?

You can select an element from the DOM using:
1. getElementById("id")– By ID.
2. getElementsByClassName("class")- By class name.
3. getElementsByTagName("tag")– By tag name.
4. querySelector("selector") – By CSS selector (first match).
5. querySelectorAll("selector")– By CSS selector (all matches).

What is event delegation in the context DOM and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events for its child elements. Instead of adding multiple listeners to each child, the event bubbles up to the parent, which can capture and handle it.

Why it's useful:

Improves performance: Fewer event listeners are needed, especially in dynamic lists. Handles dynamically added elements: The parent can manage events for elements added later without needing new listeners.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using the DOM in various ways. To change an element's attributes, you can use methods like `setAttribute(name, value)` to set a specific attribute, `getAttribute(name)` to retrieve its value, and `removeAttribute(name)` to delete it. For styling, you can directly modify CSS properties using `element.style.property`, such as `element.style.backgroundColor = "blue"` to change the background color. Additionally, the `classList` property allows you to manage classes easily: you can add classes with `element.classList.add("className")`, remove them with `element.classList.remove("className")`, and toggle their presence with `element.classList.toggle("className")`. This flexibility enables dynamic and interactive web designs.
